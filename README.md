Written in Javascript on React CRA

To run, use the following command: 

```js
npm install // get all the dependencies
npm start
```

#### Sorting Algorithms
- Bubble Sort
- Insertion Sort
- Merge Sort
- Heap Sort
- Quicksort (with Lomuto Partition Scheme)
- Quicksort (with Hoare Partition Scheme)
